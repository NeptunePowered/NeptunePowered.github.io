What is Neptune?

Neptune is an implementation of CanaryLib for both 'vanilla' Minecraft and Minecraft Forge. It aims to be the best and most lightweight implementation of CanaryLib.

What about CanaryMod?

We think that CanaryLib is a fantastic API, and felt the current implementation inadequately matches. We plan to make Neptune the faster and most lightweight implementation availiable.
